This is the calendar made from scratch for this project. It is rendered by javascript and adjusts automatically for each month selected. All the days are clickable to create new events and once an event is created this is also clickable. Clicking on a day will take you to the add event page, automatically inserting the date for you. To make handling dates in javascript a little easier I made us of the open source Luxon library.
