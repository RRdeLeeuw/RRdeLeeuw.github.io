This is a calendar that is rendered by javascript and adjusts automatically for each month selected. To make handling dates in javascript a little easier this calendar makes use of the open source Luxon library.

This calendar is not fully functional. The full functionality for this calendar (adding an item, changing an item, the items rendering in the calendar etc.) a database must be connected to store user login data and calendar event data. This is just a static version for demo on GitHub.
